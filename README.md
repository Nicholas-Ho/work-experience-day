# Alexa Marketing Playground

A simple starter project for work experience day.

## What this repo does
- Keeps all UI in a single `index.html` file.
- Deploys automatically to GitHub Pages via GitHub Actions on every push to `main`.

## How students use it
1. Open `index.html`.
2. Ask Copilot to make a small visual/content change.
3. Commit and push to `main`.
4. GitHub Actions deploys the updated page to GitHub Pages.

## One-time repo setting
In GitHub repo settings, set **Pages** source to **GitHub Actions**.
